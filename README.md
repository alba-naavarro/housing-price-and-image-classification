# 🧠 Deep Learning with TensorFlow: Neural Networks and CNNs

## Overview

This repository contains a **Deep Learning** project developed in **Python** using **TensorFlow** and **Keras**. It demonstrates the implementation, training, and evaluation of both **Dense Neural Networks (DNNs)** and **Convolutional Neural Networks (CNNs)** for regression and image classification tasks.

The notebook follows a complete deep learning workflow, including data preprocessing, model architecture design, training, hyperparameter selection, and performance evaluation.

## Projects

### 🏠 California Housing Price Prediction

A fully connected neural network is trained on the **California Housing** dataset to predict median house prices.

The workflow includes:

- Data preprocessing and normalization
- Train-test split
- Dense Neural Network implementation
- Model training and validation
- Regression performance evaluation

### 🖼️ CIFAR-10 Image Classification

A Convolutional Neural Network (CNN) is developed using the **CIFAR-10** dataset to classify images into ten categories.

The project includes:

- Image preprocessing
- CNN architecture design
- Model training
- Performance evaluation
- Prediction on unseen images

## Datasets

- [CIFAR-10](https://cave.cs.toronto.edu/kriz/cifar.html)
